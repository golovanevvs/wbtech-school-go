# Calendar - Микросервис календаря событий

HTTP-сервис для управления календарем событий с веб-интерфейсом.

Протестировать сервис можно по ссылке:
[https://incompletely-elemental-moonfish.cloudpub.ru/calendar_web-client/](https://incompletely-elemental-moonfish.cloudpub.ru/calendar_web-client/)

## Описание

Микросервис "Календарь событий" предоставляет систему для создания, редактирования и управления событиями в календаре. Включает в себя веб-интерфейс на Next.js и backend сервер на Go.

## Технологии

### Frontend (calendar_web-client)

- **Next.js 14** - React фреймворк
- **TypeScript** - типизированный JavaScript
- **Material-UI** - библиотека UI компонентов
- **FullCalendar** - компонент календаря
- **React** - библиотека для создания UI

### Backend (calendar_main-server)

- **Go** - язык программирования
- **Gin** - HTTP роутер
- **PostgreSQL** - база данных
- **Docker** - контейнеризация
- **Telegram Bot** - для уведомлений через Telegram
- **Email Service** - для уведомлений через e-mail
- **Prometheus** - мониторинг и метрики

## API Endpoints

### События

- `GET /events/month?year=2024&month=1` - Получить события на месяц
- `GET /events/day?date=2024-01-15` - Получить события на день
- `GET /event/{id}` - Получить событие по ID
- `POST /event/create` - Создать новое событие
- `PUT /event/{id}` - Обновить событие
- `DELETE /event/{id}` - Удалить событие

### Здоровье системы

- `GET /health` - Проверка состояния сервера

## Функциональность

### Основные возможности календаря

1. **Просмотр событий**:
   - Месячный, недельный и дневной вид
   - Навигация по датам
   - Отображение всех событий на выбранный период

2. **Создание событий**:
   - Форма создания события с валидацией
   - Поддержка полноценных и кратких событий
   - Установка времени начала и окончания

3. **Редактирование событий**:
   - Перетаскивание событий для изменения времени
   - Изменение размера события для изменения продолжительности
   - Немедленное отображение изменений в интерфейсе

4. **Система тем**:
   - Светлая и темная тема
   - Переключение через кнопку в header

### Backend возможности

1. **Фоновые воркеры**:
   - Отправка напоминаний о событиях
   - Автоматическая архивация старых событий

2. **Асинхронный логгер**:
   - Неблокирующее логирование запросов
   - Централизованная обработка логов

3. **Интеграции**:
   - Telegram бот для управления событиями
   - Email уведомления
   - Prometheus метрики
