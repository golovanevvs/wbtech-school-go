# Event Booker

Сервис бронирования с дедлайнами.

Протестировать работу сервиса можно по ссылке:

[https://incompletely-elemental-moonfish.cloudpub.ru/event-booker_web-client/](https://incompletely-elemental-moonfish.cloudpub.ru/event-booker_web-client/)

## Краткое описание

Event Booker — это веб-приложение для организации и бронирования мероприятий. Пользователи могут:

- Смотреть список доступных событий;
- Регистрироваться и входить в систему;
- Создавать мероприятия;
- Бронировать места на мероприятия;
- Получать уведомления о событиях на электронную почту и Telegram;
- Управлять своим профилем.

## Технологии

**Frontend:**

- Next.js + React + TypeScript;
- Material-UI для интерфейса;
- React Hook Form для форм.

**Backend:**

- Go + Gin;
- PostgreSQL для базы данных;
- JWT для авторизации;
- Telegram Bot API для уведомлений;
- Email уведомлений;
- Prometheus для мониторинга.

## Структура проекта

```Bash
event-booker/
├── event-booker_web-client/     # Веб-интерфейс (Next.js)
├── event-booker_main-server/    # API сервер (Go)
```

## Основные функции

### Для пользователей

- **Регистрация и вход** — создание аккаунта и авторизация;
- **Просмотр событий** — список всех доступных мероприятий;
- **Бронирование мест** — запись на интересные события;
- **Управление профилем** — изменение личных данных и настроек уведомлений;
- **История бронирований** — просмотр своих записей.

### Для организаторов

- **Создание событий** — добавление новых мероприятий;
- **Управление местами** — контроль количества доступных мест.

## Уведомления

Система поддерживает два типа уведомлений:

1. **Telegram** — сообщения в Telegram боте;
2. **Email** — письма на электронную почту.

Пользователи могут включить/выключить каждый тип уведомлений в настройках профиля.
