# Sales Tracker

Сервис для отслеживания продаж с аналитикой и агрегированием данных.

Протестировать работу сервиса можно по ссылке:

[https://incompletely-elemental-moonfish.cloudpub.ru/sales-tracker_web-client/](https://incompletely-elemental-moonfish.cloudpub.ru/sales-tracker_web-client/)

## Краткое описание

Sales Tracker — это веб-приложение для учёта и анализа продаж. Пользователи могут:

- Добавлять записи о доходах и расходах;
- Просматривать таблицу всех записей с фильтрацией и сортировкой;
- Получать аналитические данные за период (сумма, среднее, количество, медиана, перцентиль);
- Визуализировать динамику продаж в виде графиков;
- Экспортировать данные в формате CSV;
- Редактировать и удалять существующие записи.

## Технологии

**Frontend:**

- Next.js + React + TypeScript;
- Material-UI для интерфейса;
- React Hook Form для форм;
- @mui/x-charts для графиков;
- @mui/x-date-pickers для выбора дат.

**Backend:**

- Go + Gin;
- PostgreSQL для базы данных;
- SQL агрегация для аналитических запросов.

## Основные функции

### Управление записями

- **Добавление записей** — создание новых записей о доходах и расходах;
- **Просмотр записей** — таблица всех записей с возможностью сортировки;
- **Редактирование** — изменение существующих записей прямо в таблице;
- **Удаление** — удаление записей с подтверждением;
- **Фильтрация** — сортировка по различным полям (ID, тип, категория, дата, сумма).

### Аналитика

- **Периодическая аналитика** — расчёт показателей за выбранный период:
  - Общая сумма (sum);
  - Средний чек (avg);
  - Количество записей (count);
  - Медиана;
  - 90-й перцентиль.
- **Визуализация данных** — график динамики продаж по дням;
- **Экспорт данных** — скачивание записей в формате CSV.

## Типы записей

Приложение поддерживает два типа записей:

1. **Доходы** — положительные суммы;
2. **Расходы** — отрицательные суммы.

Каждая запись содержит:

- Тип (доход/расход);
- Категорию;
- Дату;
- Сумму.

## Аналитические возможности

Система предоставляет мощные возможности для анализа финансовых данных:

- **Временной анализ** — отслеживание трендов по дням;
- **Статистические показатели** — медиана и перцентиль для понимания распределения;
- **Визуальная аналитика** — интерактивные графики для быстрого понимания данных;
- **Экспорт** — возможность выгрузки данных для внешнего анализа (в формате csv).
